# pscc2014Icon

Photoshop CC 2014 Icon Resource Extractor/Packer (for Windows)

## Usage
pscc2014Icon \[-e|-p\] \<idx_file\> \[\<working_directory\>\]

## Example

* Extract icons: 
    * pscc2014Icon -e "C:\Program Files\Adobe\Adobe Photoshop CC 2014\Resources\IconResources.idx" "Work"
* Pack icons: 
    * pscc2014Icon -e "C:\Program Files\Adobe\Adobe Photoshop CC 2014\Resources\IconResources.idx" "Work"
